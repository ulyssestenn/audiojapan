How to Add More Voices

This app uses your browser + operating system text-to-speech voices. For privacy/security reasons, websites cannot install voices automatically.

Windows 10/11

  • Open Settings ‚Üí Time & language ‚Üí Language & region.
  • Add Japanese (or another language) under preferred languages if needed.
  • Open Speech settings and install additional speech packs/voices.
  • Restart the browser and click Refresh Voice List in this app.

macOS

  • Open System Settings ‚Üí Accessibility ‚Üí Spoken Content.
  • Choose System Voice ‚Üí Manage Voices‚Ä¶.
  • Download Japanese and English voices you want.
  • Reopen the browser tab and click Refresh Voice List.

ChromeOS / Linux

  • Voice availability depends on browser and installed speech engines.
  • Install language packs/voices at OS level (or browser-specific voice extensions where supported).
  • Reload this page and refresh the voice list.
